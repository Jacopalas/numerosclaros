¿Qué calcula?

Estima el Impuesto sobre la Renta de las Personas Físicas aplicando los tramos estatales vigentes. Calcula la cuota íntegra progresiva y el tipo medio efectivo resultante, desglosando cuánto pagas en cada tramo.

Cómo usar

AtributoTipoDescripción
base-imponibleNúmeroBase imponible general en euros
comunidad-autonomaTextoComunidad autónoma (para tramo autonómico)
situacion-personalTextoSituación personal (soltero, casado, familia_numerosa...)

Ejemplo de uso como Web Component:

<calc-irpf base-imponible="35000" comunidad-autonoma="madrid" situacion-personal="soltero"></calc-irpf>

Ejemplo de embed

<!-- Incluir el script de NumerosClaros -->
<script src="https://cdn.numerosclaros.com/calculadoras.js"></script>

<!-- Insertar la calculadora -->
<calc-irpf
  base-imponible="35000"
  comunidad-autonoma="madrid"
  situacion-personal="soltero">
</calc-irpf>

En adelante47%
  • Cada tramo se aplica solo a la porción de base que cae dentro de ese rango
  • El tipo medio efectivo es la cuota total dividida entre la base imponible
  • Los tramos autonómicos se suman a los estatales para calcular la cuota íntegra total
