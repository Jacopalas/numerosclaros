Retorno de Inversión (ROI)

Universal

¿Qué calcula?

Calcula la rentabilidad porcentual de una inversión comparando lo que has ganado con lo que has invertido. Es la métrica más directa para evaluar si una inversión ha merecido la pena, expresada como porcentaje sobre el capital invertido.

Cómo usar

AtributoTipoDescripción
inversionNúmeroCapital invertido en euros
retornoNúmeroValor final obtenido en euros

Ejemplo de uso como Web Component:

<calc-roi inversion="10000" retorno="13500"></calc-roi>

Ejemplo de embed

<!-- Incluir el script de NumerosClaros -->
<script src="https://cdn.numerosclaros.com/calculadoras.js"></script>

<!-- Insertar la calculadora -->
<calc-roi
  inversion="10000"
  retorno="13500">
</calc-roi>

Fórmula

ROI = (Retorno − Inversión) / Inversión × 100
  • ROI — Retorno de inversión (%)
  • Retorno — Valor final obtenido
  • Inversión — Capital inicial invertido

Un ROI positivo indica ganancia, uno negativo indica pérdida. No tiene en cuenta el factor tiempo (para eso usar TIR).
