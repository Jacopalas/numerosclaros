TIR / VAN

Universal

¿Qué calcula?

Calcula la Tasa Interna de Retorno (TIR) y el Valor Actual Neto (VAN) de un proyecto de inversión. Permite evaluar si una inversión con múltiples flujos de caja es rentable y comparar distintas alternativas de inversión entre sí.

Cómo usar

AtributoTipoDescripción
inversion-inicialNúmeroInversión inicial (valor positivo)
flujosTextoFlujos de caja separados por comas (ej: "5000,6000,7000")
tasa-descuentoNúmeroTasa de descuento anual (%)

Ejemplo de uso como Web Component:

<calc-tir inversion-inicial="50000" flujos="15000,18000,20000,22000" tasa-descuento="8"></calc-tir>

Fórmula

VAN = Σ [CFt / (1+r)t] − I0

TIR: r tal que VAN = 0
  • VAN — Valor Actual Neto
  • CFt — Flujo de caja en el periodo t
  • r — Tasa de descuento
  • t — Periodo (año)
  • I0 — Inversión inicial
  • TIR — Tasa de descuento que hace VAN = 0 (se calcula por iteración)
