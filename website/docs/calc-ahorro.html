Planificador de Ahorro

Universal

¿Qué calcula?

Calcula cuánto debes ahorrar mensualmente para alcanzar un objetivo financiero en un plazo determinado. Tiene en cuenta el ahorro inicial disponible y la rentabilidad esperada de la inversión para darte una cifra realista de ahorro mensual necesario.

Cómo usar

AtributoTipoDescripción
objetivoNúmeroCantidad objetivo en euros
plazoNúmeroPlazo en años
interesNúmeroRentabilidad esperada anual (%)
ahorro-inicialNúmeroAhorro inicial disponible en euros

Ejemplo de uso como Web Component:

<calc-ahorro objetivo="50000" plazo="10" interes="5" ahorro-inicial="5000"></calc-ahorro>

Fórmula

FV = PV(1+r)n + PMT × [((1+r)n − 1) / r]
  • FV — Valor futuro (objetivo de ahorro)
  • PV — Valor presente (ahorro inicial)
  • r — Tipo de interés mensual
  • n — Número de meses
  • PMT — Aportación mensual necesaria (lo que se resuelve)

Se despeja PMT de la fórmula para obtener la aportación mensual requerida.
