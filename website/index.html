Ver calculadoras GitHub

Cómo usar

1

CDN

Añade el script y usa el componente directamente:

<script src="https://cdn.jsdelivr.net/npm/numerosclaros@latest/dist/numerosclaros.min.js"></script>

<nc-hipoteca></nc-hipoteca>
2

npm

Instala el paquete e impórtalo en tu proyecto:

npm install numerosclaros
import 'numerosclaros'
3

Google Sheets

Usa las fórmulas directamente en tu hoja de cálculo:

=NC_HIPOTECA(200000, 25, 2.5)
