Cómo usar

1

CDN — una línea de HTML

Añade el script y usa el componente directamente en cualquier web:

<script src="https://cdn.jsdelivr.net/npm/numerosclaros@latest/dist/numerosclaros.min.js"></script>

2

npm

Instala el paquete e impórtalo en tu proyecto:

npm install numerosclaros

// Usa en tu HTML:
// <nc-hipoteca></nc-hipoteca>

Google Sheets

Embed en tu web

Copia y pega. Funciona en cualquier web: WordPress, Webflow, HTML estático...

Cada calculadora es un Web Component estándar. Solo necesitas añadir el script una vez y usar las etiquetas HTML que quieras. Sin frameworks, sin dependencias, sin configuración.

<!-- 1. Añade el script (una sola vez) -->
<script src="https://cdn.jsdelivr.net/npm/numerosclaros@latest/dist/numerosclaros.min.js"></script>

<!-- 2. Usa las calculadoras que necesites -->
<nc-hipoteca
  importe="200000"
  plazo="25"
  interes="2.5">
</nc-hipoteca>

<nc-irpf
  bruto="35000"
  comunidad="madrid">
</nc-irpf>
Aquí se renderizaría <nc-hipoteca>
